# 🕒 Big Typography Web Stopwatch & Clock

사용자 경험(UX)을 극대화한 대형 텍스트 기반의 웹 스탑워치 및 시계 어플리케이션입니다. 
시인성을 최우선으로 설계되었으며, 브라우저 환경에서 정확한 시간 측정을 보장합니다.

## ✨ Key Features
- **Gigantic Display**: `clamp()` 함수를 활용하여 어떤 화면에서도 가득 차는 반응형 타이포그래피 구현.
- **Smart Mode Switching**: 스탑워치 미사용 시 현재 시간을 보여주는 시계 모드로 자동 전환.
- **High Accuracy**: `requestAnimationFrame`과 타임스탬프 델타 계산을 통한 정밀한 시간 측정.
- **Modern UI**: 다크 모드 기반의 미니멀한 디자인과 고정폭 폰트(`tabular-nums`)를 통한 레이아웃 안정성 확보.

## 🛠 Tech Stack
- **Frontend**: HTML5, CSS3 (Modern Flexbox/Grid), JavaScript (ES6+)
- **Performance**: Zero-dependency, Vanilla JS
